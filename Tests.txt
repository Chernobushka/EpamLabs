https://www.catfootwear.com/

1) Добавление товара в корзину
Шаги:
	-Зайти на главную страницу сайта
	-Перейти в раздел "Men"
	-Перейти в раздел "Boots"
	-Перейти на страницу выбранного товара. Например https://www.catfootwear.com/US/en/second-shift-work-boot/018462632905.html
	-Выбрать цвет, размер и ширину
	-Нажать кнопку "ADD TO CART"
Результат: товар с выбранным цветом, размером и шириной добавится в корзину, а также появится окно с описанием заказа
Если товара нет в наличии, то появится кнопка "NOTIFY WHEN AVAILABLE"

2) Добавление товара в корзину без указания размера и ширины
Шаги:
	-Зайти на главную страницу сайта
	-Перейти в раздел "Men"
	-Перейти в раздел "Boots"
	-Перейти на страницу выбранного товара. Например https://www.catfootwear.com/US/en/second-shift-work-boot/018462632905.html
	-Нажать кнопку "ADD TO CART"
Результат: Если не выбран размер и ширина, то появится сообщение "Please select a size and a width before adding to the cart."
Если выбрана ширина но не выбран размер, то появится сообщение "Please select a size before adding to the cart."
Если выбран размер но не выбрана ширина, то появится сообщение "Please select a width before adding to the cart."

3) Выбот товара, которого нет в наличии
Шаги:
	-Зайти на главную страницу сайта
	-Перейти в раздел "Men"
	-Перейти в раздел "Boots"
	-Перейти на страницу выбранного товара
	-Выбрать цвет, размер, ширину
	-Нажать кнопку "NOTIFY WHEN AVAILABLE"
Результат: Если пользователь не вошел в аккаунт, то появится окно с сообщением "The following item is out of stock" и кнопка "LOGIN/REGISTER"
Если пользователь вошел в аккаунт, то появится окно с сообщением "The following item is out of stock" и кнопка "NOTIFY WHEN AVAILABLE". 
После нажатия кнопки "NOTIFY WHEN AVAILABLE" появится окно с сообщением
"This item has been successfully added to your Notify When Available List."

4) Добавление товаров на сумму больше 99$
Шаги
	-Выбираем товар стоимотью более 99$. Например https://www.catfootwear.com/US/en/excavator-superlite-waterproof-nano-toe-work-boot/44904M.html?dwvar_44904M_color=P91218#cgid=mens-boots&start=1
Результат: Express доставка будет выбрана автоматически и ее стоимость изменится с 15$ до 0$

5) Удаление товара из корзины пр наличии других товаров
В корзине находятся 2 товара стоимостью 84$ и 160$
Шаги:
	-Перейти в корзину
	-Удалить один из товаров. Например стоимостью 160$
Результат: тип доставки сменится с Express на Standart. Общая стоимость заказа уменьшится до 84$. Стоимость Express доставки изменится с 0$ до 15$

6) Использование несуществующего промокода
Шаги:
	-Перейти в корзину
	-В поле "Apply Promo Code" ввсети промокод "123123"
	-Нажать кнопку "APPLY"
Результат: появится сообщение "Coupon Code "123123" is unknown."

7) Регистрация нового пользователя с некорректными данными
Шаги:
	-Нажать на кнопку "Account"
	-В появившемся окне нажать кнопку "CREATE AN ACCOUNT NOW"
	-Вводим данные:
	First Name: Степан
	Last Name: Arhipov
	Zip Code: 4879894564
	Email Adress: stepoan456gmail.com
	Confirm Email: stepoan456gmail.ru
Результат: Если введен неправильный zip code, то появится сообщение "Zip Code is not valid"
Если есть ошибка в структуре адреса электронной почты, то появится сообщение "The email address is invalid."
Если поля "Email Adress" и "Confirm Email" не совпадают, то появится сообщение "Does not match email" возле поля "Confirm Email"
Если имя и фамилия введены на разных языках, то сообщения об ошибке не появится
Если все данные введены корректно то появится страница с сообщением "Thank you for registering"

	Type: Visa

	Month: 01-JAN
